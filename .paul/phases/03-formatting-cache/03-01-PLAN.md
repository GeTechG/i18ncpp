<objective>
## Goal
Add a result cache for `formatNumber`, `formatPrice`, and `formatDate` that returns previously computed strings on repeated calls with identical inputs, and invalidates automatically when locale or formatting configuration changes.

## Purpose
In game loops and UI frames, the same formatted values are requested every frame. Caching avoids redundant number/currency/date string construction — the most CPU-intensive formatting paths in the library.

## Output
- Modified `include/i18ncpp.h` — cache member, invalidation hooks, cache helpers
- Modified `src/i18ncpp.cpp` — cache lookup/store in format methods, invalidation in mutators
- Modified `tests/i18n_test.cpp` — cache hit, invalidation, and correctness tests
</objective>
