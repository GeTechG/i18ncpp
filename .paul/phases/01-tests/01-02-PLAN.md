<acceptance_criteria>

## AC-1: English plural rules
```gherkin
Given locale is "en" with plural key having "one" and "other" forms
When trPlural is called with count=1 and count=5
Then count=1 returns "one" form, count=5 returns "other" form
```

## AC-2: Russian plural rules (one/few/many/other)
```gherkin
Given locale is "ru" with plural key having "one", "few", "many", "other" forms
When trPlural is called with count=1, 2, 5, 21, 11, 111
Then correct Slavic plural forms are selected (1→one, 2→few, 5→many, 21→one, 11→many, 111→one)
```

## AC-3: French plural rules (count < 2 = one)
```gherkin
Given locale is "fr" with plural key having "one" and "other" forms
When trPlural is called with count=0 and count=1
Then both return "one" form (French: count < 2 = one)
```

## AC-4: Polish, Czech, Arabic plural rules
```gherkin
Given locales "pl", "cs", "ar" with appropriate plural forms
When trPlural is called with representative counts
Then correct plural forms are selected per each language's rules
```

## AC-5: Pluralization with interpolation
```gherkin
Given a plural key with placeholders like "You have {0} items"
When trPlural is called with count and additional params
Then count is prepended as {0} and additional params fill {1}, {2}, etc.
```

## AC-6: Positional and unnamed interpolation
```gherkin
Given translation "Hello {0}, welcome to {1}" and "Hello {}, welcome to {}"
When tr is called with ["Alice", "Wonderland"]
Then both produce "Hello Alice, welcome to Wonderland"
```

## AC-7: Named and formatted interpolation
```gherkin
Given translation "%{name} has %<score>.d points"
When translate is called with {name: "Alice", score: 42.7}
Then result is "Alice has 42 points"
```
